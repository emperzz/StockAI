## 🚀 快速启动

### 1. 环境准备
```bash

# 安装依赖（已完成）
pip install -r my_agent/requirements.txt
```

### 2. 启动服务
```bash
# 启动前端服务
python start_frontend.py
```

### 3. 访问界面
打开浏览器访问：**http://localhost:7860**

### 常见问题
1. **端口占用**: 使用 `lsof -i :7860` 查看并释放端口
2. **依赖缺失**: 运行 `pip install -r my_agent/requirements.txt`
3. **数据获取失败**: 检查网络连接和股票代码

### 调试方法
- 查看控制台日志
- 运行测试脚本: `python test_frontend.py`
- 检查服务状态: `ps aux | grep start_frontend`
